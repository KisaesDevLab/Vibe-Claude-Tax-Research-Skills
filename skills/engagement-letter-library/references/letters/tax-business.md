# Business tax engagement letter (1120 / 1120-S / 1065 / 706 / 709 / 990)

Required elements per AICPA Tax Section practice aid + AICPA SSTS
+ FTC Safeguards Rule. Adapt to specific entity and form.

## Required elements

- [ ] Returns covered (federal + state) and tax year.
- [ ] Entity type (C-corp / S-corp / Partnership / Estate /
      Gift / Nonprofit).
- [ ] Reliance-on-client-records disclaimer.
- [ ] FTC Safeguards Rule data-security clause.
- [ ] §7216 disclosure consent for third-party use.
- [ ] Penalty / interest exposure framework.
- [ ] Multistate nexus and apportionment scope.
- [ ] Limitation of liability / indemnification (state overlay).
- [ ] Fee arrangement.
- [ ] Termination provisions.

## Practitioner notes

- For 706/709 estate/gift returns, separate engagement letter
  recommended due to special complexity (basis step-up, GST,
  closely-held valuation).
- For 990 nonprofit returns, separate engagement letter
  recommended due to public-disclosure requirements and
  conflict-of-interest sensitivities.
- Multistate nexus is fact-specific; engagement letter should
  delineate states clearly to avoid scope creep.
- For S-corp engagements, owner returns are NOT in scope unless
  separately engaged.
- For partnership engagements with §754 elections, basis-tracking
  is in scope only if specifically negotiated.
