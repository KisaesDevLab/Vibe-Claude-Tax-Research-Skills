# Bookkeeping / write-up engagement letter

For bookkeeping or write-up services. Critical scoping required
when client is also an attest client (AICPA Code §1.295 nonattest-
service implications).

## Required elements

- [ ] Specific bookkeeping scope (data entry, bank rec, GL
      maintenance, payroll posting, period-close, etc.).
- [ ] Statement that engagement is NOT an audit, review, or
      compilation.
- [ ] §1.295 four-condition affirmation if client is also attest.
- [ ] Management retains responsibility for books.
- [ ] FTC Safeguards Rule clause.
- [ ] Confidentiality.
- [ ] Out-of-scope work language.
- [ ] Fee, termination.

## Practitioner notes

- §1.295 is the gatekeeper: if client is also attest, the four
  conditions must be documented BEFORE bookkeeping starts.
- Routine entries: posting authorized by client, recurring,
  formula-based.
- Non-routine entries: management judgment required (write-offs,
  unusual accruals, valuation reserves, lease classification).
  Firm cannot post non-routine entries on attest clients without
  client management approval and review.
- If firm acts as outsourced controller / CFO: management
  participation; impairs attest independence.
- For non-attest-only clients, §1.295 does not apply; routine
  business-engagement principles govern.

## Authority

- AICPA Code §1.295 (nonattest services): cite as `AICPA_Code`,
  `binding_on_member`.
- AR-C §70 / §80 / §90: if any preparation/compilation/review
  output is anticipated, cite separately.
- FTC Safeguards Rule: regulatory (if tax return preparation is
  also engaged).
