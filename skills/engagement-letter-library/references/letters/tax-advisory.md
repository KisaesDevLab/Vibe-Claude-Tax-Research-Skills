# Tax advisory engagement letter (planning, no return preparation)

Required elements per AICPA Tax Section practice aid +
Circular 230 §10.37 (written advice). For tax planning,
consulting, or advisory engagements that do NOT include return
preparation.

## Required elements

- [ ] Specific advisory scope (planning area, tax year, specific
      transactions).
- [ ] Deliverable (memo / spreadsheet / oral advice).
- [ ] Circular 230 §10.37 written-advice standard adherence.
- [ ] Reliance on client representations.
- [ ] FTC Safeguards Rule data-security clause.
- [ ] Limitation of scope (does NOT include implementation, return
      preparation, or audit defense).
- [ ] Fee arrangement (contingent fees prohibited under §1.510
      with narrow exceptions).
- [ ] Termination provisions.

## Practitioner notes

- For "covered opinions" under prior Circular 230 (now §10.37
  reasonable-practitioner standard), no specific marketed-tax-
  shelter language is required, but the reasonable-practitioner
  standard applies to all written advice.
- For listed-transaction or reportable-transaction advisory
  scopes, Form 8886 disclosure recommended; if firm is the
  "material advisor," Form 8918 disclosure may be required.
- For state-specific advisory (PTET elections, residency
  planning, multistate apportionment), reference per-state files
  in `tax-research-state-income/references/states/`.
- For multi-jurisdictional or international advisory, route to
  `tax-research-international` skill for treaty / GILTI / FTC /
  FBAR considerations.
