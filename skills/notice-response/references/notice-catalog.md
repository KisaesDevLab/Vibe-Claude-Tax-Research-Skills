# IRS notice catalog

Verify current notice formats and deadlines via the IRS Notice
look-up at https://www.irs.gov/individuals/understanding-your-irs-notice-or-letter
or https://www.irs.gov/payments before responding.

## Categorization for response strategy

| Category | Strategy |
|---|---|
| Math-error notice (CP10–CP13) | Verify calculation; dispute via §6213(b)(2) within 60 days if math is wrong. |
| Underreporter (CP2000 / CP2501) | Substantiate the omission or accept; CP2000 is NOT a stat notice, but CP3219A is the resulting stat notice. |
| Examination report (Letter 525 / 915 / 950) | Decide: agree (sign waiver), partial agree, or appeal to Appeals. |
| Statutory Notice of Deficiency (Letter 3219) | File Tax Court petition within 90 days. JURISDICTIONAL. |
| TFRP (Letter 1153) | Protest to Appeals within 60 days; engage criminal counsel if §7202 / §7203 indications. |
| Lien filing (Letter 3172) | CDP request within 30 days for procedural and substantive review. |
| Levy notice (Letter 1058 / CP90) | CDP request within 30 days. |
| Collection (CP14 / CP501 / CP503 / CP504) | Address: pay, installment, OIC, or CNC. |
| Identity-theft (CP01) | Form 14039 if not already filed. |
| Math-error refund adjustment (CP12) | Reconcile / accept; dispute within 60 days if math wrong. |

## Common practitioner traps

- Treating CP2000 as a stat notice (it's not — but CP3219A IS).
- Missing the 90-day Tax Court deadline because of a math error
  in the receipt date.
- Filing CDP request after the 30-day window expires
  (jurisdictionally fatal for that posture; equivalent CDP
  available for next levy / lien notice).
- Filing a Letter 1153 protest after 60 days; Appeals refuses to
  consider untimely protests.
- Failing to sign Form 872 when the SOL is about to expire and
  signing is in the client's interest.
- Failing to verify that prior representation arrangement (Form
  2848 POA) is on file with the IRS for the relevant tax periods.

## Pre-response checklist

- [ ] Notice ID and date confirmed.
- [ ] Deadline calendared.
- [ ] Underlying return retrieved.
- [ ] Form 2848 POA on file for the period.
- [ ] Penalty exposure quantified.
- [ ] Substantive-position research path identified.
- [ ] FTA / reasonable-cause screened.
- [ ] CDP rights preserved if applicable.
- [ ] Statute-of-limitations posture confirmed.
- [ ] Routing to substantive skills documented.
