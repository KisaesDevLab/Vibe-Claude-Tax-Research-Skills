# IRC Title 26 — Subtitle map

26 U.S.C. (Internal Revenue Code) is organized into Subtitles
A–K. Use this map to navigate from a topic to the section number
range, or from a section number to its Subtitle.

## How to use this map

For a section-number lookup:
1. Identify the Subtitle / chapter / subchapter from the map.
2. Use the URL pattern
   `https://uscode.house.gov/view.xhtml?req=granuleid:USC-prelim-title26-section[number]`
   (e.g., `section199A` for § 199A).
3. Verify text with USLM canonical source.

For a topical lookup:
1. Match the topic to a section range.
2. Browse the relevant subchapter for the specific section.
3. Use the Popular Name Tool to identify any Public Laws that
   amended the section recently.
