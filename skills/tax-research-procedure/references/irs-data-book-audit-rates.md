# IRS Data Book — audit rates and procedural trends

## Source

IRS Data Book: https://www.irs.gov/statistics/soi-tax-stats-irs-data-book

## Authority status (CRITICAL)

The IRS Data Book is `authority_type: IRSPub` /
`weight: persuasive_non_authority` ONLY.

It is useful for:
- Setting client expectations on realistic audit / collection
  exposure.
- Demonstrating in a malpractice-defense file that a recommendation
  aligned with administrative trend data.
- Identifying high-audit-rate categories to inform client-facing
  risk discussions.

The Data Book MUST NOT serve as the closest authority for:
- A confidence-band assignment.
- A procedural recommendation.
- Any return-position support.

## Common Data Book tables relevant to tax procedure

### Audit rates by income / entity type

The Data Book reports historical examination rates ("coverage
rates") by income tier and entity type:
- Individual returns by AGI tier.
- Sole-proprietor Schedule C returns.
- Partnership returns (Form 1065).
- S-corp returns (Form 1120-S).
- Large-corporation returns (Form 1120, asset tier).
- Estate / gift returns.
- Foreign-information returns.

### Coverage-rate trend

Aggregate individual audit rates have declined substantially over
the past decade due to IRS budget constraints. Recent IRA
appropriations (Pub. L. 117-169) have funded enforcement
expansion; subsequent budget actions have moderated. Verify the
most-recent Data Book for current trend data.

### Examination outcomes

Tables also report:
- Percentage of audits resulting in change vs no change.
- Average dollar adjustment by examination type.
- Closing-agreement vs unagreed rates.

### Collection actions

Data Book reports:
- Number of liens filed.
- Number of levies issued.
- Installment-agreement counts.
- Offer-in-compromise counts and acceptance rates.

## Practitioner uses

### Client communication

"Your S-corp's 2023 examination rate was approximately X%
according to the IRS Data Book. While statistics don't predict
your individual case, this provides administrative-trend
context."

### Engagement-letter risk disclosure

"Examination rates for partnerships of your size and complexity,
per the IRS Data Book, were approximately X%."

### Engagement-quality file

Document that audit-risk assessment was informed by current Data
Book trend data; this supports a malpractice-defense narrative
that the practitioner exercised reasonable diligence.

## Cautions

- Data Book figures lag by 1-2 years; current-year trends may
  differ substantially.
- IRA-funded enforcement expansion is changing the audit
  landscape; trend data from the 2018-2022 budget environment may
  not predict the 2024+ environment.
- Categorical statistics do not predict individual-case outcomes.
- The Data Book does not include criminal-investigation referrals
  or correspondence-only audits in the same buckets; read the
  table headers carefully.
